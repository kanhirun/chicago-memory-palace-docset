Unroll Recursion

You have a recursive procedure that you wish to express iteratively.

Mechanics

  1. Replace the conditionals in the base case with an `until` loop (red)
  2. If that's absent, use a `while` loop but with all the conditions inversed.
  3. Underneath the while conditions, move the recursive statements there
  4. Replace the recursive statements with variable assignment
  5. Finally, move the base case return to the very bottom
